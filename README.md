# MyGithubApplication

My Github Application is a repository tracker app through which we can save and track our favourite/concerned repositories. First add a new repository and save it. Then we can view different branches and their related commits. Also we can view open issues related to that repository. We can also view that repository on custom tabs and can share to other different social platforms.
